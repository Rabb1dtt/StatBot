# Эталонный профиль футболиста по позициям — система метрик и весов для оценки

## Методология

Система оценки строится на сравнении реальных показателей игрока (per 90 минут) с бенчмарком эталонного игрока на его позиции. Бенчмарк отражает **медиану топ-25% исполнителей** в топ-5 европейских лигах — то есть уровень, к которому нужно стремиться, но который достижим у качественных игроков.

Метрики разделены на 4 уровня влияния:

| Уровень | Вес в итоговой оценке | Описание |
|---|---|---|
| 🔴 **Ключевые** | 35–40% | Определяют суть роли, без них игрок не соответствует позиции |
| 🟠 **Важные** | 25–30% | Существенно влияют на эффективность, сильно дифференцируют уровень |
| 🟡 **Средние** | 20–25% | Дополняют профиль, важны в контексте командной системы |
| 🟢 **Низкие** | 10–15% | Дают нюанс, сигнализируют о стиле, но не определяют качество |

Итоговая оценка за каждый блок нормируется по Z-score относительно позиционного пула. Финальный рейтинг — взвешенная сумма по блокам (0–10).

## 1. Вратарь (GK)

Голкипер — единственная позиция с принципиально иным набором метрик.

### 🔴 Ключевые метрики (40%)
| Метрика | Описание | Эталон (per 90 / сезон) |
|---|---|---|
| **PSxG +/−** | Разница между ожидаемыми пропущенными (PSxG) и реальными. Главный показатель качества сейвов | +0.10 за игру |
| **Save %** | % отражённых ударов от всех ударов в створ | ≥ 73% |
| **Big Chance Save %** | % спасений в моментах с вероятностью гола > 40% (ключевые моменты) | ≥ 65% |

### 🟠 Важные метрики (25%)
| Метрика | Описание | Эталон |
|---|---|---|
| **Goals Prevented** | Голы предотвращены выше среднего вратаря | ≥ +3 за сезон |
| **Launch %** | % ударов ногой (vs розыгрыш короткими пасами) — определяет стиль | контекстуально |
| **Crosses Claimed %** | % успешно перехваченных или выбитых прострелов | ≥ 55% |

### 🟡 Средние метрики (20%)
| Метрика | Описание | Эталон |
|---|---|---|
| **Passes Completed (>40m) %** | Точность длинных передач — важно для команд с высокой линией | ≥ 55% |
| **Pass Completion % (short)** | Точность розыгрыша в ногах — для команд с билд-апом снизу | ≥ 85% |
| **Sweeper Actions per 90** | Выходы за штрафную для перехватов/подборов | ≥ 1.5 |

### 🟢 Низкие метрики (15%)
| Метрика | Описание |
|---|---|
| **Clean Sheets %** | Командная метрика, не только заслуга вратаря |
| **Penalties Saved %** | Малая выборка, высокая случайность |
| **Действия ногами — AvgLen** | Средняя длина удара ногой — стилистический маркер |

## Шкала итогового рейтинга

| Оценка | Уровень |
|---|---|
| 9.0 – 10.0 | Мировой топ (Haaland, Vinicius Jr., Rodri) |
| 8.0 – 8.9 | Элита (ротация топ-клубов, потенциал основы) |
| 7.0 – 7.9 | Высокий уровень (середина таблицы топ-лиг) |
| 6.0 – 6.9 | Среднее (хороший игрок для лиги B) |
| < 6.0 | Ниже стандарта для топ-уровня |

**Ключевые принципы:**
- Все метрики нормируются **per 90 минут** для устранения влияния игрового времени
- Минимальная выборка — **10+ полных матчей** (900+ минут) для статистической значимости
- Ключевые метрики при резком отклонении вниз (< 30th percentile) применяют **штрафной коэффициент ×0.85** к итоговому баллу
- Метрики командно-зависимые (xA, progressive passes) корректируются на **стиль команды** (владение, прессинг-интенсивность)
